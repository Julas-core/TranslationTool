# Language Translation Tool

A real-time translation web app built as part of my AI internship at [CodeAlpha](https://codealpha.tech). Supports 14 languages including Amharic and Arabic.

🔗 **[Live Demo](https://translate.vercel.app)** · 💻 **[GitHub](https://github.com/julas-core/translationtool)**

---

![screenshot](/assets/homepage.png)

---

## Features

- Translate between 14 languages in real time
- Amharic, and Tigrinya support (East African languages)
- RTL text direction for Arabic
- Swap source and target languages with one click
- Copy translation to clipboard
- Character counter
- Keyboard shortcut — `Ctrl + Enter` to translate
- Fully responsive

## Tech Stack

- HTML, CSS, JavaScript
- [MyMemory Translation API](https://mymemory.translated.net)

## Supported Languages

English · French · Spanish · Portuguese · German · Italian · Arabic · Amharic · Hindi · Chinese · Japanese · Korean · Russian · Swahili

## Getting Started

No setup needed — just open `index.html` in a browser.

```bash
git clone https://github.com/julas-core/translationtool
cd TranslationTool
open index.html
```
